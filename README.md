# Breast Cancer Diagnosis Analysis

## Overview
Exploratory data analysis of the Breast Cancer Wisconsin 
Diagnostic dataset using PySpark and Pandas to identify 
patterns that distinguish malignant from benign tumours.

## What This Project Covers
- Loading and exploring medical datasets using PySpark
- Data cleaning and validation
- Statistical analysis of key diagnostic features
- Visualizing diagnosis class balance using Matplotlib
- Extracting meaningful diagnostic indicators

## Tools Used
- Python
- PySpark
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Key Findings
- Dataset contains 569 patient records
- 37% Malignant cases and 63% Benign cases
- Identified key features that distinguish malignant 
  from benign tumours including radius mean, 
  texture mean and perimeter mean

## How to Run
1. Install requirements:
pip install pyspark pandas numpy matplotlib scikit-learn

2. Open breast_cancer.ipynb in Jupyter Notebook
3. Run all cells from top to bottom
